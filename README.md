> Generate stylish, unique, and engaging Instagram bios in just one click! ✨

Welcome to the **Instagram Bio Generator**, a sleek and interactive web application designed to help you craft the perfect Instagram bio. Whether you want something cute, savage, motivational, funny, or aesthetic, this tool has you covered!

## ✨ Key Features

- **🎭 Multiple Categories:** Choose from Cute, Savage, Motivational, Funny, Aesthetic, or Custom styles.
- **👁️ Live Instagram Preview:** See exactly how your generated bio will look on a simulated Instagram profile before you copy it.
- **🌗 Dark/Light Mode:** Toggle between beautifully designed dark and light themes for comfortable viewing anytime.
- **📋 One-Click Copy:** Easily copy your new bio to the clipboard with a single click.
- **✍️ Editable Bios:** Not perfectly satisfied? Edit the generated bio directly within the app before copying.
- **📈 Trending Bios Section:** Browse through a slider of currently trending bios for instant inspiration.
- **📱 Responsive Design:** Works flawlessly on desktops, tablets, and mobile devices.
- **🎨 Modern Glassmorphism UI:** Enjoy a premium visual experience with glass-like cards, beautiful gradients, and smooth animations.

## 🚀 How to Use

1. **Clone or Download** this repository to your local machine.
2. Open the `index.html` file in any modern web browser.
3. **Type your Name** in the provided input field (e.g., your real name or username).
4. **Select a Category** from the dropdown menu to match your vibe.
5. Click the **"Generate Bio"** button to magically create a new bio.
6. Scroll down to the **Insta Preview** to see how it looks on a real profile layout.
7. If you want to tweak it, toggle **Customize Bio** and make your edits.
8. Click **"Copy to Clipboard"** and paste it directly into your Instagram profile!

## 🛠️ Technologies Used

- **HTML5:** Core structure and semantics.
- **CSS3:** Styling, glassmorphism (`backdrop-filter`), CSS variables, and responsive layouts.
- **JavaScript (Vanilla):** DOM manipulation, bio generation logic, clipboard integration, and theme toggling.
- **FontAwesome:** Scalable vector icons.
- **Google Fonts (Inter):** Clean and modern typography.
